📦 Addon Description
=========================

This addon adds an access management system for **Q-menu** and **C-menu** in Garry’s Mod.  
Administrators can open a special menu 🖥️ that displays a list of all players 👥, and with simple ✅❌ buttons, grant or revoke their access to Q and C menus.  
All changes are saved in a JSON file 💾 and automatically loaded when the server restarts.
